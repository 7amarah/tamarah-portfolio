# Tamarah Portfolio (Next.js)

Modern, SEO-optimized portfolio built with Next.js, Tailwind CSS, and Framer Motion.

## 🚀 Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Formspree (contact form)

## 📦 Features
- SEO optimized pages
- Responsive design
- Projects showcase
- Contact form (no backend required)
- Fast performance (SSG + SSR)

## 🛠️ Setup

```bash
npm install
npm run dev
